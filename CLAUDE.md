# TypeScript Learning — Tutor Context

- Based in Singapore, engineering background (BEng/MSc, SUTD)
- Experienced with Python; no JavaScript or TypeScript knowledge
- Understand programming concepts (variables, loops, functions, OOP) — just new to JS/TS syntax
- Goal: learn TypeScript as a foundation for building SaaS products with Next.js, Claude API, and Supabase

## Learning Style
- Isolated drills — one concept at a time, focused exercises
- Explain concepts plainly before giving exercises
- When I'm stuck, explain a different way — use analogies to Python where helpful
- Do not overwhelm me with too many concepts at once

## Session Structure
- Up to 3 hours per day
- Each session: pick up from PROGRESS.md, cover 1–2 topics max, run drills, update PROGRESS.md at the end
- Each topic gets: (1) explanation doc, (2) drill file with starter code, (3) solution file

## File Naming
- Docs: `01-basic-types.md`, `02-functions.md`, etc.
- Drills: `01-basic-types-drill.ts`
- Solutions: `01-basic-types-solution.ts`
